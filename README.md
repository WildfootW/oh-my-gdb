# oh-my-gdb

## command
* `init-pwndbg`
    * load pwndbg/pwndbg@stable
* `init-glibc-source`
    * Set directory for glibc source
* `init-angelheap`
    * load scwuaptx/Pwngdb/angelheap (manual track)
* `init-pwngdb`
    * load scwuaptx/Pwngdb (manual track)
